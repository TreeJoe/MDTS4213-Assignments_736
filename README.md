# Predictive-Analysis-Practical-Projects
- This repository contains datasets and analysis projects done as part of the Predictive Analysis course.
- The .csv file is the data in hand.
- Problem Set 1: An Introduction.Rmd file is the Rmarkdown of the analysis according as the problem set 1.
- Problem Set 1: An Introduction.pdf file is the PDF output of the corresponding Rmd file.
- Problem Set 2 - Linear Regression.Rmd file is the Rmarkdown of the analysis according as the problem set 2.
- Problem Set 2 - Linear Regression.pdf file is the PDF output of the corresponding Rmd file.

### Boston
